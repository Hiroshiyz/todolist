# Todolist

a simple web application for todo list built with Flask and SQLite3

## Features

- add new task
- Delete task
- Mark task as complete
- Local storage support

## Tech Stack

-HTML
-Flask
-SQLite3

## How to use

1. clone the repo:
   ```bash
   git clone https://github.com/Hiroshiyz/todolist.git
   cd todolist
   ```
2. Open file:
   ```bash
   python app.py
   ```
